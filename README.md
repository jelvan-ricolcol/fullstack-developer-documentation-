## Verification policy

Documentation must cite official standards bodies, vendor documentation, or mature security references. Examples include MDN, WHATWG, W3C, IETF RFCs, OWASP, NIST, Cloudflare Docs, GitHub Docs, AWS Docs, Docker Docs, Kubernetes Docs, OpenTelemetry, Prometheus, Stripe Docs, and OpenAI Docs.

When official sources change, the linked source takes precedence over repository prose. Contributors should update affected pages with the source URL, the date checked, and the command or review used to validate examples.
